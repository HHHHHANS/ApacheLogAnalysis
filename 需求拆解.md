# 总体框架模块：
+ data端（数据获取/格式转换）
+ analysis端（解析）
+ report端（输出报表）
+ tunnel模块（封装消息队列）

data端 --(tunnel模块)--> analysis端 --(tunnel模块)--> report端

# 各模块说明
### agent端：
+ 从本地/服务器获取日志
+ 日志内容安全性检测/解析
+ 读取不同类型的日志文件
+ 对于本地日志文件做大小限制

### tunnel模块：
+ 限制日志内容传输速度/大小
+ 可对transfer两端对接的模块进行配置
+ 

### analysis端：
+ 接收可处理的日志字符串
+ 将日志内容进行解析
+ 

### report端:
+ 从tunnel模块接收解析后的日志信息组成相应形式的报告
